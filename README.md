# Client–Server UDP Application (Qt + C++)

Кроссплатформенное клиент-серверное приложение, реализующее обмен данными по UDP с графическим интерфейсом на Qt.  
Проект демонстрирует работу с сетевыми сокетами, управление подключениями, обработку пакетов, ограничение частоты запросов и визуализацию состояния взаимодействия между клиентами и сервером.

---

## ✨ Возможности

### 🖥 Сервер
- Принимает данные от нескольких клиентов одновременно
- Обрабатывает UDP-пакеты по заданному протоколу
- Хранит информацию о клиентах:
  - IP-адрес
  - количество отправленных пакетов
  - количество ошибок
  - текущее состояние
- Ограничивает частоту запросов клиента в соответствии с конфигурацией
- Загружает лимиты из limits.json
- Отображает сведения в удобном GUI
- Сортирует список клиентов по количеству ошибок

### 💻 Клиент
- Отправляет пакеты на сервер по UDP
- Позволяет включать / отключать отправку
- Показывает лог отправленных сообщений и ответов
- Валидирует параметры подключения
- Визуально отображает состояние соединения
